# Backend Environment Variables

This document describes all environment variables used by the backend service.

## Required Variables

### `OPENAI_API_KEY`

- **Description**: Your OpenAI API key for accessing GPT-4 models
- **Required**: Yes
- **Example**: `sk-proj-abc123...`
- **How to get**:
  1. Go to [OpenAI Platform](https://platform.openai.com/)
  2. Sign up or log in
  3. Navigate to API Keys section
  4. Create a new API key
